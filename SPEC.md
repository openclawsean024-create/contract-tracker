## 1. 問題陳述

### 1.1 目標使用者

| 族群 | 規模 | 痛點 |
|---|---|---|
| 中小企業主 | ~15 萬 | 合約散落在 email/紙本/雲端，到期未續簽造成違約風險 |
| 接案 SOHO 族 | ~30 萬 | 客戶合約條款各自不同，難以快速審視風險 |
| 房東/二房東 | ~5 萬 | 多份租約到期提醒、押金/維修責任條款對照 |
| 業務/業務主管 | ~8 萬 | 需要快速查詢「這份合約裡有沒有提到 OOO 法條」 |

### 1.2 為什麼不做替代方案

- **紙本歸檔**：搜尋困難、易遺失、合約到期不會自動提醒
- **雲端硬碟（Google Drive）**：能存但無法自動對照法條、無到期提醒
- **商用 CLM 系統（ContractWorks/Icertis）**：年費 30 萬起，對微型企業太貴
- **我們的解法**：純前端 + 預載 100 條台灣常用法條摘要 + 風險評分引擎，零月費

---

## 2. 解決方案

### 2.1 核心價值主張

> 「上傳合約 PDF，自動標出潛在風險條款 + 到期前 30 天提醒。」

### 2.2 使用者流程

1. 上傳合約 PDF（純前端解析，無上傳雲端）
2. 系統擷取關鍵條款（保密/違約/智慧財產/期限）
3. 自動與預載的 100 條法條摘要對照，標示風險（綠/黃/紅）
4. 使用者設定到期提醒（30/60/90 天前）
5. 月底儀表板顯示：即將到期合約清單 + 風險評分排行

## 3. 功能清單

### 3.1 MVP（必做）

- [ ] 合約 PDF 上傳 + 文字擷取（pdf.js）
- [ ] 預載 100 條台灣常用法條摘要（民法/勞基法/著作權法/消保法/個資法）
- [ ] 法條對照引擎（自動標示合約中可能違反的條文）
- [ ] 風險評分（綠/黃/紅 三級）
- [ ] 到期提醒（30/60/90 天前推播）
- [ ] 範本庫（10 種常見合約範本：服務/聘僱/租賃/NDA 等）

### 3.2 v2（加值）

- [ ] AI 條款解釋（用 LLM 解釋法律術語為白話文）
- [ ] 合約版本比對（修改前後差異）
- [ ] 多用戶協作（律師/企業主共用）
- [ ] 雲端同步（目前 localStorage 限制單機）

### 3.3 明確不做

- 線上簽署（交給第三方服務如 DocuSign）
- 法律諮詢（明確聲明僅供參考，不取代律師）
- 跨境合約（先做台灣法域）
- 仲裁調解（超出工具範圍）

## 4. 技術棧

| 層 | 選擇 | 理由 |
|---|---|---|
| 前端 | React 18 + Vite + TypeScript | 與既有架構一致 |
| PDF 解析 | pdf.js (Mozilla) | 業界標準、純前端 |
| 法條資料 | 預載 JSON 檔案 | 法條公開資料整理、定期更新 |
| 全文搜尋 | FlexSearch | 輕量、支援中文（自訂 tokenizer） |
| 資料持久化 | localStorage |
| 部署 | Vercel |

## 6. 風險與決策

| 風險 | 等級 | 緩解 |
|---|---|---|
| 法條更新（條文修正/廢止） | 🔴 高 | 預載 JSON 版本化、定期從全國法規資料庫同步 |
| AI 給出錯誤法律建議導致糾紛 | 🔴 高 | 明確免責聲明 + 鼓勵使用者諮詢專業律師 |
| PDF 解析失敗（掃描檔/加密檔） | 🟠 中 | 提供手動貼上文字 fallback；v2 加 OCR |
| 個資疑慮（合約含個資） | 🟠 中 | 全 localStorage 處理、不上傳雲端、隱私頁說明 |
